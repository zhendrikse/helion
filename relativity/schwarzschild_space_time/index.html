Important result

Section titled “Important result”

A stable circular orbit only exists if r>3Mr > 3Mr>3M!

For:

  • r=3Mr = 3Mr=3M → photon sphere (instable light orbit)
  • r<3Mr < 3Mr<3M → no stable circular orbit possible
